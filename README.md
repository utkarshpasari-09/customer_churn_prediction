# Customer Churn Prediction

An end-to-end machine learning project to predict customer churn using a trained model and a Streamlit web application.

## Features
- Data preprocessing and feature scaling
- Supervised ML model (SVM)
- Handles class imbalance
- Interactive Streamlit web app
- Clean, deployment-ready UI

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
